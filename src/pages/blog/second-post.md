# マークダウンの使い方

この記事では、ブログ記事でマークダウンを使用する方法について説明します。

## 基本的な書式

マークダウンでは以下のような書式が使えます：

- **太字** や *イタリック*
- [リンク](https://example.com)
- リスト（番号付き・番号なし）
- > 引用

## コードブロック

function hello() {
  console.log("Hello, World!");
}
